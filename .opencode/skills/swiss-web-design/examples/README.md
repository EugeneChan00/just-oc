# Swiss Web Design Examples

This folder is a workspace for collecting reference material when working on Swiss design projects.

## How to Use

When starting a project, the agent should:

1. Search online for Swiss web design references
2. Save interesting website screenshots or HTML snippets here
3. Use these as calibration for the right level of minimalism

## Folder Structure

- `./` — downloaded website files, HTML snippets, saved pages
- `./images/` — screenshots, mood boards, design reference images
